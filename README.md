# EShopMicroservices
This project uses a microservices architecture built on .NET 8 and C# 12, featuring ASP.NET Web API, Docker, RabbitMQ with MassTransit, gRPC, YARP API Gateway, PostgreSQL, Redis, SQLite, SQL Server, Marten, and EF Core. It applies CQRS, MediatR, DDD, and Clean/Vertical Architecture for scalability and maintainability
